# Gaming Battle Platform - Admin Panel

This is the administration panel of the Gaming Battle Platform built with Next.js 15, TypeScript, and Redux Toolkit.

## Project Structure

- `src/app`: Page components and routing.
- `src/components`: Reusable presentation and layout components.
- `src/store`: State management config, services (RTK Query), and slices.
- `src/hooks`: Global custom hooks.
